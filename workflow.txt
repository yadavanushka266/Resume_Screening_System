
## Features

* Upload resumes in PDF, DOCX, or TXT format
* Automatic text extraction from uploaded resumes
* Resume preprocessing using NLP techniques
* Predicts the most relevant job category
* Displays prediction confidence
* Simple and user-friendly Streamlit interface
* Fast and accurate resume classification

## Machine Learning Pipeline

1. Resume Upload
2. Text Extraction
3. Text Cleaning and Preprocessing
4. TF-IDF Vectorization
5. Feature Extraction
6. K-Nearest Neighbors (KNN) Classification
7. Job Category Prediction
8. Display of Predicted Role





## How It Works

* The user uploads a resume.
* The system extracts text from the uploaded document.
* NLP preprocessing removes unwanted characters, punctuation, and stop words.
* The processed resume is transformed into numerical features using TF-IDF.
* The trained KNN classifier predicts the most appropriate job category.
* The predicted category is displayed instantly through the Streamlit interface.

## Conclusion

The Resume Screening System demonstrates how Machine Learning and Natural Language Processing can streamline the recruitment process by automatically categorizing resumes into relevant job domains. By using TF-IDF for feature extraction and the K-Nearest Neighbors algorithm for classification, the system provides an efficient, scalable, and reliable solution for resume screening, reducing manual effort while improving hiring accuracy.
